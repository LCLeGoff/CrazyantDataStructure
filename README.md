# Python virtual env
Virtual envs are created in `~/.virtualenvs` and can be created with :

`mkvirtualenv -p python3 <virtualenv_name>`

When you want to use a virtualenv just do:

`workon <virtualenv_name>`

To stop using it:

`deactivate`
